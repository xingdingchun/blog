# Python迭代器

## 迭代器概念

### 可迭代对象

生成器都是迭代器，迭代器不一定都是生成器。

list,tulpe,dict,string都是Iterable(可迭代对象)

### for循环过程

1. 调用可迭代对象iter方法，返回一个迭代对象
2. 不断调用迭代器next()方法
3. 处理StopTteartion异常

循环列表的过程：
<pre>
for i in [1,2,3,4,5,6,]:
    print(i)

# 内部的过程是使用iter方法
# 不断执行netx()方法
# 并处理最后循环完成的异常
</pre>

## 迭代器

满足条件：
1. iter 方法
2. next 方法

菲波那切数列
<pre>
def fobi(x):
    n,before,after=0,0,1
    while n < x:
        before,after = after,before + after
        n += 1
    return after
print(fobi(7))
</pre>
