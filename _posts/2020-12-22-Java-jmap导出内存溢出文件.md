# Java jmap导出内存溢出文件

## 使用jmap导出内存溢出文件
    jmap -dump:format=b,file=heap.hprof pid(进程号)

    format=b 文件格式
    file=heap.hprof 文件名称
    9 进程号码

    jmap -dump:format=b,file=heap.hprof 9
