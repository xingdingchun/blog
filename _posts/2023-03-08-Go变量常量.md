# Go变量常量



### 变量的来历



> 程序运行过程中的数据都是保存在内存中，我们想要在代码中操作某个数据时就需要去内存上找到这个变量，但是如果我们直接在代码中通过内存地址去操作变量的话，代码的可读性会非常差而且还容易出错，所以我们就利用变量将这个数据的内存地址保存起来，以后直接通过这个变量就能找到内存上对应的数据。



### 变量的类型



> 变量(variable)的功能是存储数据。不同的变量保存的数据类型可能会不一样。经过半个多世纪的发展，编程语言已经基本形成了一套固定的类型，常见变量的数据类型有：整型、浮点型、布尔型等。
>
> 



Go中的变量



> Go中的每个变量都有自己的类型，并且变量必须经过声明才能开始使用，而且声明以后必须使用。Go中的变量名由字母、数字、下划线组成，其中首个字符不能是数字，Go语言中关键字和保留字不能作为变量名。Go中变量声明后没有初始化的值为空。





### Go中变量声明

```go
// 声明的变量未赋值默认为空
package main

import "fmt"

func main() {
	// Go中变量声明
	var username string
	fmt.Println(username)
	//声明的变量没有赋值默认为空

}
// 正确的声明变量的方式
package main

import "fmt"

func main() {
	// Go中变量声明
	var username string = "chad"
	fmt.Println(username)
	/*
		chad
	*/
}
// 错误变量定义
package main

import "fmt"

func main() {
	// Go中变量声明
	var 1username string = "chad"
	fmt.Println(username)
	/*
	 syntax error: unexpected literal 1, expected name
	*/

}
// 正确的变量定义，但是不推荐
package main

import "fmt"

func main() {
	// Go中变量声明
	var m_a string = "chad"
	fmt.Println(m_a)
	/*
       chad
	 */

}
// if关键字不能作为变量定义
package main

import "fmt"

func main() {
	// Go中变量声明
	var if string = "chad"
	fmt.Println(m_a)
	/*
    syntax error: unexpected if, expected name
	 */

}
```



### Go中不同变量定义方法



```go
// Go变量先定义在赋值
package main

import "fmt"

func main() {
	// Go中变量声明
	var username string
	username = "Hello,Chad"
	fmt.Println(username)
	/*
		Hello,Chad
	*/
}
// 定义即赋值
package main

import "fmt"

func main() {
	// Go中变量声明
	var username string = "定义即赋值Hello,chris"
	fmt.Println(username)
	/*
		定义即赋值Hello,chris
	*/
}
// 定义变量类型可以省略
package main

import "fmt"

func main() {
	// Go中变量声明
	var username = "定义变量类型可以省略"
	fmt.Println(username)
	/*
		定义变量类型可以省略
	*/
}
```



### Go中变量声明才可以使用不能重复声明



```go
// Go中正确声明变量
package main

import "fmt"

func main() {
	// Go中变量声明
	var username = "chad"
	var age = 20
	var sex = "男"

	fmt.Println(username, age, sex)
	fmt.Printf("姓名是:%v,年龄是:%v,性别是:%v", username, age, sex)
	/*
			chad 20 男
	        姓名是:chad,年龄是:20,性别是:男
	*/
}
// 变量不能重复声明但是可以重复赋值
package main

import "fmt"

func main() {
	// Go中变量声明
	var username = "chad"
	var age = 20
	var sex = "男"
	// 重复赋值变量
	username = "chris"

	fmt.Println(username, age, sex)
	fmt.Printf("姓名是:%v,年龄是:%v,性别是:%v", username, age, sex)
	/*
	   chris 20 男
	   姓名是:chris,年龄是:20,性别是:男
	*/
}
```



### Go中一次声明多个变量



```go
// 一次声明多个变量(问题是类型定义不灵活)
package main

import "fmt"

func main() {
	// Go中变量声明
	var username, age, sex string
	username = "chad"
	age = "18"
	sex = "女"

	fmt.Println(username, age, sex)
	fmt.Printf("姓名是:%v,年龄是:%v,性别是:%v", username, age, sex)
	/*
		chad 18 女
	    姓名是:chad,年龄是:18,性别是:女
	*/
}
// 改进一次声明多个变量
package main

import "fmt"

func main() {
	// Go中变量声明
	var (
		username string
		age      int
		sex      string
	)

	username = "chad"
	age = 18
	sex = "男"

	fmt.Println(username, age, sex)
	fmt.Printf("姓名是:%v,年龄是:%v,性别是:%v", username, age, sex)
	/*
	   chad 18 男
	   姓名是:chad,年龄是:18,性别是:男
	*/
}
// 直接赋值多个变量定义
package main

import "fmt"

func main() {
	// Go中变量声明
	var (
		username string = "chad"
		age      int    = 18
		sex      string = "男"
	)

	fmt.Println(username, age, sex)
	fmt.Printf("姓名是:%v,年龄是:%v,性别是:%v\n", username, age, sex)
	fmt.Printf("姓名的类型是:%T,年龄的类型是:%T,性别的类型是:%T", username, age, sex)
	/*
	chad 18 男
	姓名是:chad,年龄是:18,性别是:男
	姓名的类型是:string,年龄的类型是:int,性别的类型是:string
	*/
}
```

	*/

		姓名是:chad,年龄是:18,性别是:男
		姓名的类型是:string,年龄的类型是:int,性别的类型是:string
	*/
}
// 端变量定义方式只能定义局部变量不能定义全局变量

### 匿名变量



> 在使用多重赋值时，如果想要忽略某个值，可以使用匿名变量(anonymous variable)，匿名变量用一个下划线_表示；
>
> 匿名变量不占用命名空间，不会分配内存，所以匿名变量之间不存在重复声明



```go
// 定义函数并返回值
package main

import "fmt"

func getUserinfo() (string, int, string) {
	return "auther", 18, "男"

}

func main() {
	var username, age, sex = getUserinfo()
	fmt.Printf("姓名是:%v,年龄是:%v,性别是:%v\n", username, age, sex)
	fmt.Printf("username类型是:%T,age类型是是:%T,sex类型是:%T", username, age, sex)
	/*
		姓名是:auther,年龄是:18,性别是:男
	    username类型是:string,age类型是是:int,sex类型是:string
	*/
}
// 定义匿名变量
package main

import "fmt"

func getUserinfo() (string, int, string) {
	return "auther", 18, "男"

}

func main() {
	var username, _, _ = getUserinfo()
	fmt.Printf("姓名是:%v\n", username)
	fmt.Printf("username类型是:%T", username)
	/*
	姓名是:auther
	username类型是:string
	*/
}
```
